Bài 6: Kiểm tra mảng số nguyên

    Mô tả:
        Viết hàm với tham số là mảng số nguyên.
        Hàm thực hiện kiểm tra.
    Yêu cầu:
        Nếu mảng chỉ chứa số chẵn thì trả về 1.
        Nếu mảng chỉ chứa số lẻ thì trả về 0.
        Ngược lại (chứa cả chẵn, cả lẻ) thì trả về -1.
    Ví dụ:
        Input: 2, 8, 12, 4. -> Output: 1
        Input: 1, 7, 3, 11 -> Output: 0
        Input: 1, 2, 9, 4 -> Output: -1
    Kỹ năng liên quan:
        Định nghĩa và sử dụng hàm
        Làm việc với mảng (duyệt qua các phần tử)
        Kiểm tra tính chẵn/lẻ của số (toán tử %)
        Câu lệnh điều kiện.
