Bài 1: Kiểm tra mảng tăng dần

    Mô tả:
        Nhập một mảng số từ bàn phím với độ dài n (với n <= 20).
        Hiển thị mảng sau khi nhập.
        Kiểm tra xem mảng vừa nhập có phải là mảng tăng dần hay không.
    Yêu cầu:
        Nếu mảng là tăng dần thì hiển thị "OK".
        Nếu mảng không tăng dần thì trả về "NO".
    Ví dụ:
        Input: 0,2,4,6,8 -> Output: "OK"
        Input: 5,10,15,20 -> Output: "OK"
        Input: 2,4,7,9 -> Output: "OK"
        Input: 1,5,4,9 -> Output: NO
    Kỹ năng liên quan:
        Nhập liệu cơ bản, làm việc với:
        Mảng (khai báo, nhập phần tử, hiển thị)
        Vòng lặp
        Câu lệnh điều kiện
        Kiểm tra điều kiện trên các phần tử mảng liên tiếp.
